## primary
 - pc 작업 기준 : 1920px
 - 폰트사이즈 px 기준
 - 이미지는 % 기준
 
 ***주석***
 - 큰 섹션: = x 5
 - 중간 섹션: = x 3
 - 구분선: - x 5
 - 기타: 기본

***이미지 네이밍 규칙***
- 콘텐츠이름-순번.확장자 (appleImg-1.png)
- 긴 이름은 케밥표기법으로 단어를 줄여서 사용 (apple-img-1.png x, appleImg-1.png o)
